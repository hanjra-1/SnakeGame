# Snake Game

A classic implementation of the Snake game using HTML, CSS, and plain JavaScript.

## Description

This is a simple, browser-based Snake game where the player controls a snake to eat food and grow longer. The game ends if the snake collides with the walls or with itself.

## Features

-   Classic snake gameplay
-   Score tracking
-   Game over when the snake hits a wall or itself
-   Simple and clean UI

## How to Play

1.  **Clone the repository or download the files.**
2.  **Open the `index.html` file in your web browser.**
3.  **Use the Arrow Keys (Up, Down, Left, Right) to control the snake.**
4.  **Eat the food (the red squares) to grow longer and increase your score.**
5.  **Avoid running into the walls or the snake's own body.**

## Technologies Used

-   **HTML5**
-   **CSS3**
-   **JavaScript (ES6)**

## File Structure

-   `index.html`: The main HTML file containing the game board structure.
-   `Style.css`: The CSS file for styling the game.
-   `script.js`: The JavaScript file containing the game logic.
-   `README.md`: This file.
